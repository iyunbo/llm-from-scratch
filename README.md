# Build a LLM from Scratch 🧠

My implementation of a GPT-style Large Language Model from scratch in PyTorch, following [Sebastian Raschka's book](https://www.manning.com/books/build-a-large-language-model-from-scratch).

## Why

Understanding LLMs by building one — no shortcuts, no frameworks, just raw PyTorch.
