LLM From Scratch

A complete GPT-style language model built from first principles in PyTorch, following Sebastian Raschka's Build a Large Language Model (From Scratch).

What This Project Covers

Every layer of the GPT architecture is implemented from scratch — from BPE tokenization through multi-head attention to instruction-tuned chat. Each chapter builds on the last, with runnable demos and detailed explanations.

Chapter 2

Text Data & Tokenization

BPE tokenizer, token & positional embeddings, sliding-window dataloader.

 Chapter 3

Attention Mechanisms

Simple, causal, and multi-head attention — built incrementally from raw matrix math.

 Chapter 4

GPT Model Architecture

Transformer blocks, GELU, LayerNorm, residual connections, and text generation.

 Chapter 5

Pretraining

Full training loop with warmup, cosine LR decay, gradient clipping, and checkpointing.

 Chapters 6 & 7

Fine-tuning

Transfer learning for classification, then instruction tuning for an interactive chatbot.

Quick Start

# Clone and set up
git clone https://github.com/iyunbo/llm-from-scratch.git
cd llm-from-scratch
python -m venv .venv && source .venv/bin/activate
pip install torch tiktoken numpy

# Pretrain on sample text
python -m ch05.pretrain

# Chat with the instruction-tuned model
python -m ch07.chat

Key Takeaways

InsightWhy It Matters
LLMs are stacked transformer blocksThe core architecture is embeddings + attention + feedforward, repeated N times
Next-token prediction is enoughThis single self-supervised objective forces models to learn deep language understanding
Scale is architecture-agnosticSame code works from 128-dim toys to GPT-2 XL (1.5B params)
Transfer learning is powerfulPretrained weights adapt to classification and instruction-following with minimal extra training
